# docs/

Esta carpeta está reservada para los 5 documentos pedagógicos del curso original
(`01_interfaces.md` a `05_jugador.md`), que viven en el Google Drive del usuario y
no están versionados aquí.

No es crítico tenerlos en el repo: la **arquitectura completa** (clases, métodos,
constructores, plan por fases y bugs conocidos a corregir) ya está extraída y
documentada en `../CLAUDE.md`, que es la fuente de verdad para la implementación.

Si en algún momento se vuelcan los `.md` originales en esta carpeta, sirven como
material de referencia adicional, pero no son requeridos para construir el juego.
