# Space Invader 👾☄️

Clon de Space Invaders en Python + pygame-ce, construido como proyecto de curso.

## Requisitos
- Python 3.10+ (probado en 3.14)
- pygame-ce (drop-in compatible con pygame; soporta Python 3.14)

## Instalación
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Ejecutar
```bash
python main.py
```

> **Estado: WIP — Fase 1.** Por ahora solo abre la ventana 800×600 y dibuja el HUD (vidas, nivel, balas). El jugador, enemigos, balas y colisiones llegan en fases siguientes.
